# Solitare Card Games
A Windows Forms .NET application using cards.dll from Windows XP.  Original authored in 2007.  Still works in Windows 10 when compiled as 32-bit application.

## List of games
* Aces Up
* Beleaguered Castle
* Blackjack
* Calculation
* Canfield
* Corners
* Freecell
* Golf
* Klondike
* Midnight Oil
* Monte Carlo
* Osmosis
* Reno
* Royal Cotillion
* Towers
